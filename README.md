EsAoN (Extraction and Optimization Engine)
O EsAoN é um motor de busca e filtragem de dados de alta performance, projetado para realizar a mineração cirúrgica de informações na web. Ele serve como a camada de entrada para o ecossistema VECNA.

 Funcionalidades Principais
Scraping Inteligente: Coleta automatizada de dados utilizando técnicas para evitar bloqueios e garantir a integridade da informação.

Filtragem de Ruído: Algoritmos que eliminam leads irrelevantes, garantindo que apenas dados de alta qualidade cheguem à camada de processamento.

Modularidade: Capaz de se adaptar a diferentes nichos de mercado com ajustes mínimos de configuração.

 Stack Técnica
Python (Core engine)

Playwright / Selenium (Browser automation)

BeautifulSoup4 (Parsing de HTML)

Regex (Limpeza e padronização de dados)

Status: Em fase de otimização de performance para grandes volumes de dados.
